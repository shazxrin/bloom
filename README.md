## Setup

### Web Application

```bash
gradle npmInstall
```

### Server

```bash
gradle build
```

## Development

### Web Application

```bash
gradle devWebapp
```

### Server

```bash
gradle bootRun
```

## Building

### Web Application

```bash
gradle bundleWebapp
```

### Server

```bash
gradle bootBuildImage
```
